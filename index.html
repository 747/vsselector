同じ字でも Unicode に微妙な字形の区別が登録されている漢字・絵文字・数学記号・モンゴル系文字などを簡単に検索・入力できるようにしたものです。コピーした文字は一般のソフトでも利用できますが、違いを目で見るためには対応したフォントが必要な場合があります。詳しい情報はこちらなどを参照してください。

使い方
下書き欄
ここに文字を入力したり、検索結果や異体字セレクタを挿入できます。欄内の文字の一覧が下に表示されるので、不可視の文字でも選んで削除できます。
セレクタ類一覧
字形を制御する（多くは不可視の）文字の一覧です。文字をクリックすると下書き欄のカーソル位置に挿入されます。
検索
文字や Unicode 番号を入力すると、登録のある異体字がある場合は結果が表示されます。一度に複数文字入力して一括検索ができます。また、絵文字の入力補助として :東京: のように半角コロンで囲むことで、連想される絵文字の候補が出ます。
  • ⬆挿入 を押すと、上の欄に字が挿入されます。
  • コピー を押すと、文字をコピーして他の場所に貼り付けできるようにします。
  • 表示欄にあるものが実際の文字で、画像欄にあるのがサンプル画像です。
動作環境

ブラウザは Internet Explorer 10 以上。IE9（Windows 7 付属）では一部動作に支障が出ます。 その他のブラウザ (Firefox, Chrome, Opera...) はなるべく最新のバージョンに更新してください。

Windows 7 や Mac OS X 10.6 より古い OS は、そのままでは異体字の表示が困難です。

CLDR: 42

A utility for searching & inputting variants of the same character that have different appearances defined in Unicode, coming from scripts like Chinese (Han), Emoji, Math, Mongolian etc. Copied strings as such are platform-agnostic, but may need specific font support to see the intended glyph. For how it works, recommend reading the detailed explanation by Unicode or BabelStone.

How to use
Memo Area
You can directly type characters in or insert from search results and Selectors Picker. All individual code points the text area currently contains are shown below, that helps you to check and delete invisible characters.
Selectors Picker
The list of (mostly invisible) glyph control characters. Click to paste the character into the caret position in Memo Area.
Search Function
Input characters and Unicode code points (can be mixed) to show all existing registered variants for each. Emoji suggestion is activated with the format enclosed by colons, such as :tokyo:.
  • Insert ⬆ : insert the character into the text area above.
  • Copy : copy the character data that can be pasted to elsewhere.
  • The Output column shows the actual character, and the Image column a sample image.
